# Mouse Trail Particles

An interactive canvas animation built with Flask and JavaScript.

Move your mouse to generate glowing particles.
Particles fade smoothly and leave a dynamic trail.

## Features

- Mouse movement generates particles
- Smooth fade-out animation
- Glowing visual effects
- Multiple drawing modes:
  - Point
  - Line
  - Ring
- Press 1 / 2 / 3 to switch modes
- Press Space to cycle modes

## Tech Stack

- Python (Flask)
- HTML5 Canvas
- JavaScript
- CSS

## How to Run

```bash
pip install flask
python app.py
